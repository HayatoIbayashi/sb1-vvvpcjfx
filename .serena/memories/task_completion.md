# タスク完了時のチェックリスト

- 依存追加やコード変更後は以下を実行:
  - Lint: `pnpm lint` で静的解析を通す
  - ビルド確認: `pnpm build` が成功すること
  - 動作確認: `pnpm dev` でローカル実行し主要画面を確認
- 追加で望ましいこと:
  - タイプエラーがないこと (開発/ビルド時に検出)
  - `.env` の変更があれば影響範囲を明記
  - ルーティング/認証 (Cognito) のフローに破壊的変更がないことを確認
- テスト: 現状テスト設定は未整備 (Vitest等の導入を検討)。導入後は `pnpm test` を通すこと。
